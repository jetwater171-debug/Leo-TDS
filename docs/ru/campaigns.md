# Кампании

## Что такое кампания

Кампания — это верхний уровень настройки. Внутри кампании хранятся:

- domains
- white settings
- black settings
- scripts
- postbacks
- statistics

## Основные действия

Из dashboard можно:

- создать кампанию
- дублировать кампанию
- переименовать кампанию
- удалить кампанию

## Домены

Кампания определяется по домену. Домены указываются без `http://` или `https://`.

Поддерживаются wildcard-значения, например:

- `*.example.com`
