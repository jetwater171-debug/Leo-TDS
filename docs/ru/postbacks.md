# Postbacks

## Что покрывает этот раздел

Postbacks в продукте отвечают за:

- приём внешних статусов
- обновление внутренних статусов и payout
- отправку исходящих S2S postbacks

## Внутренние статусы

Основные статусы:

- Lead
- Purchase
- Reject
- Trash

![Раздел postbacks](../assets/screenshots/postbacks-settings-overview.png)

## S2S postbacks

Для каждой записи можно задать:

- URL
- method
- список events
