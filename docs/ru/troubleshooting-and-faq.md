# Troubleshooting и FAQ

## Не получается войти в админку

Проверьте:

- `adminPassword`
- `adminDomain`
- блокировку по rate limit

## Пустая статистика

Проверьте:

- date range
- timezone
- filters
- наличие кликов в кампании

## Не обновляются конверсии по postback

Проверьте:

- `clickid`
- `status`
- payout
- mapping статусов
