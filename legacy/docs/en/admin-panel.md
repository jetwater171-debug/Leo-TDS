# Admin Panel

## Main Pages

- `admin/index.php` — campaigns dashboard
- `admin/campsettings.php` — campaign settings
- `admin/statistics.php` — statistics tables
- `admin/clicks.php` — click logs

## Main Capabilities

- manage campaigns
- configure traffic logic
- inspect statistics
- inspect click data
- manage folders and files
