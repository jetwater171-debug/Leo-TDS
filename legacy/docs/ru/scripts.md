# Scripts

## Для чего нужен раздел Scripts

Этот раздел управляет дополнительным поведением страниц и переходов.

## Основные опции

- backfix
- replace prelanding
- replace landing
- images lazy load
- next redirect rules
- submit redirect rules
- event tracking

![Раздел scripts](../assets/screenshots/scripts-settings-overview.png)

## Event tracking

Система может инжектировать отслеживание событий, например:

- scroll thresholds
- time thresholds

Потом эти значения появляются в статистике как `event.*`.
