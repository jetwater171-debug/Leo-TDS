# Click views

## Доступные представления

Страница `admin/clicks.php` поддерживает:

- Allowed
- Blocked
- Leads
- Trafficback

## Что можно делать

- переключать view
- менять columns
- сохранять filters
- искать по User ID и Click ID для некоторых view
- экспортировать данные
